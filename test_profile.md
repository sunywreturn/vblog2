# 个人主页功能测试说明

## 已实现的功能

### 前端组件
1. **UserProfile.vue** - 个人信息页面
   - 显示用户基本信息（用户名、昵称、邮箱、注册时间、用户状态、用户角色）
   - 支持编辑个人资料（昵称、邮箱）
   - 显示用户统计信息（文章总数、总浏览量、草稿数量）
   - 支持头像更换功能

2. **MyArticles.vue** - 个人文章列表页面
   - 显示用户的所有文章
   - 支持按状态筛选（全部、已发表、草稿、回收站）
   - 支持按标题搜索
   - 支持文章操作（查看、编辑、删除、还原）

### 后端API
1. **UserController** 新增接口：
   - `PUT /user/profile` - 更新用户资料
   - `GET /user/statistics` - 获取用户统计信息

2. **ArticleController** 新增接口：
   - `GET /article/user/{userId}` - 获取指定用户的文章列表
   - `DELETE /article/delete/{aid}` - 彻底删除文章

3. **UserService** 新增方法：
   - `updateUserNickname()` - 更新用户昵称
   - `getUserStatistics()` - 获取用户统计信息

4. **ArticleService** 新增方法：
   - `getArticlesByUserId()` - 根据用户ID获取文章列表
   - `getArticleCountByUserId()` - 获取用户文章总数

### 数据库映射
1. **UserMapper.xml** - 新增用户统计查询
2. **ArticleMapper.xml** - 新增用户文章查询

## 路由配置
- `/userProfile` - 个人主页
- `/myArticles` - 我的文章
- 在顶部用户下拉菜单中添加了相应的跳转链接

## 测试步骤

1. **启动项目**
   - 后端：访问 http://localhost:8081
   - 前端：访问 http://localhost:8080

2. **登录系统**
   - 使用任意账号登录

3. **访问个人主页**
   - 点击顶部用户下拉菜单中的"个人主页"
   - 验证个人信息显示是否正确
   - 测试编辑资料功能

4. **访问我的文章**
   - 点击顶部用户下拉菜单中的"我的文章"
   - 验证文章列表显示
   - 测试筛选和搜索功能
   - 测试文章操作功能
